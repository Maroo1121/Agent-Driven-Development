# Changelog

All notable changes to the ADD Framework will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] — 2025-XX-XX

### Added

- Initial release of the ADD Framework.
- Manifesto with 10 founding principles.
- The ADD Cycle: Scope → Frame → Constrain → Execute → Verify → Consolidate.
- Six standard artifacts: Agent Brief, Context Pack, Constraint Sheet, Validation Checklist, Execution Log, Failure Review.
- Maturity Model (L1–L4).
- Non-Negotiable Rules (Five Nevers).
- Comparison with TDD, DDD, and BDD.
- CLI Workflow guide.
- Anti-Patterns catalog (10 patterns).
- Quick Start guide (ADD Lite).
- Six ready-to-use templates.
- Two worked examples (simple feature + complex refactor with failure).
- Scaffold script for project initialization.
- English and French documentation.
